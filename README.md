# 🌸 Syntera

> **Em desenvolvimento** 🚧

A Syntera é uma nova versão e uma reimaginação de um projeto anterior chamado **NyxNote**, desenvolvido com o objetivo de oferecer uma plataforma mais completa, moderna e inteligente para organização dos estudos.

Este projeto está sendo reconstruído do zero, com uma nova arquitetura, novo design e novas funcionalidades.

## 🚧 Status do projeto

A Syntera está em desenvolvimento ativo.

Atualmente não existe previsão de conclusão do projeto. Ele será desenvolvido conforme houver disponibilidade de tempo e novas ideias forem sendo implementadas.

## ✨ Objetivo

A proposta da Syntera é centralizar diversas ferramentas utilizadas por estudantes em um único ambiente, incluindo:

- 📅 Organização de estudos
- ✅ Gerenciamento de tarefas
- ⏰ Pomodoro
- 📝 Flashcards
- 🤖 Assistente com IA
- 👥 Comunidade de estudantes
- 📊 Estatísticas de produtividade
- 🎯 Planejamento inteligente

## 🔄 Origem

A Syntera surgiu como uma evolução do projeto **NyxNote**.

O repositório da versão anterior pode ser encontrado em:

**NyxNote:**  
https://github.com/Mablys/NyxNote-app

Embora compartilhem a mesma ideia central, a Syntera foi planejada como uma nova aplicação, com foco em melhor organização do código, maior escalabilidade e uma experiência de uso mais completa.

## 🛠️ Tecnologias

Atualmente o projeto está sendo desenvolvido utilizando:

- HTML5
- CSS3
- JavaScript

Novas tecnologias poderão ser incorporadas ao longo do desenvolvimento.

## 📌 Aviso

Este repositório pode sofrer alterações frequentes, incluindo mudanças na estrutura do projeto, funcionalidades e interface.

Alguns recursos podem estar incompletos ou temporariamente indisponíveis durante o desenvolvimento.
